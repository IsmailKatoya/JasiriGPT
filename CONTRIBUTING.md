# Contributing to JasiriGPT (HAI-2026-007)

We welcome contributions that align with our mission of **Sovereign AI for Kenya**.

## How to Contribute
1. **Report Bugs:** Use GitHub Issues to report inaccuracies in policy retrieval.
2. **Feature Requests:** Suggest new policy documents (Gazettes, Bills) to be indexed.
3. **Code:** - Fork the repository.
   - Create a feature branch (`git checkout -b feature/NewFeature`).
   - Ensure all LLM calls remain local via Ollama.
   - Submit a Pull Request.

## Security Standards
All contributions must respect **Data Sovereignty**. No external API calls (OpenAI, Anthropic, etc.) are permitted in the RAG pipeline.
